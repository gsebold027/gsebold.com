## 🎯 Project Goals

- Build a simple but modern and accessible landing page
- Use this project to explore full-stack delivery from design to deployment
- Learn and apply tools like shadcn/ui, Docker, Argo CD, Cloudflare, and GitHub Actions
- Ensure the site supports dark/light themes and multiple languages (EN/PT)
- Host the app on Kubernetes via DigitalOcean with GitOps deployment


## 🛠 Tech Stack

| Area       | Tool / Library                        |
|------------|---------------------------------------|
| Frontend   | React + Vite                          |
| UI         | shadcn/ui (Tailwind + Radix UI)       |
| i18n       | react-i18next                         |
| Theming    | Tailwind + `class` toggling           |
| CI/CD      | GitHub Actions                        |
| Infra      | Docker, Kubernetes, Argo CD           |
| Hosting    | DigitalOcean Kubernetes               |
| CDN + SSL  | Cloudflare                            |
